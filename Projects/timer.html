Run Timer

C#
I was using an existing timer that was used to time speedruns (completing video games as fast as possible). The timer would keep track of your best segments of the run. I added a feature that would calculate the user's best possible time by adding their current time to the sum of their best segments for the rest of the run. This kind of timer functionality could also be used in a context of running a long distance, and the idea is that you want to see the best possible time if you ran your best from the current point to the end.
