Tweet Analyzer

Practical Programming in Java
Created a program that analyzed a series of tweets. It took in a file with word 'sentiments', and then was able to perform calculations on a set of tweets in order to calculate average sentiment for certain regions and tweets.
Main concepts implemented:
  • Introduction to data structures in java, ArrayList, classes, other basic types
  • Introduction to Java packages
  • Method overrides
  • Java method documentation
