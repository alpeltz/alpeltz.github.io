Natural Language Processing

Introduction to Programming and Data Structures
Implemented simple natural language processing to do post classification. The model is trained on a set of posts, each given a specific categroy. After training, new posts have their vocabulary analyzed and determine the log likelihood of the category for the posts.
Main concepts implemented:
  • Created a binary search tree and map implementation
  • Introduction to log likelihood and basic machine learning concepts
  • Basic introduction to Node data structures, with pointers to other nodes that eventually build a tree
  • BST traversal implementations (preorder, inorder)
  • Implemented custom iterator for BST
  • Introduction to recursive functions with BST, such as finding the height by recursively finding the height of the right or left subtrees
  • Dynamic memory allocation with new and delete
  • Custom map implementation with templating and using Pair types
