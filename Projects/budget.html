Budget Tracking

Practical Programming in Java
Created a mobile android application, that allows the user to track their budget. The user can add new transactions and add a category and price for each transaction. From there, they are able to view all of their categories and transactions through the app. This project involved the scripting of the application to store data as well as developing a pleasing front-end interface
Main concepts implemented:
  • Experience using Android Studio to develop an app
  • Mobile app GUI development
  • Mobile dialog alerts to inform the user of important information
  • Use of “activities” in app, which allow the app to have different ‘pages’ of sorts, and invoke new activities when clicked
  • Use of Java views to design a new page , view holders to make sure that model binding works correctly
  • Learning how Android applications can store data between sessions so that the app is not reset when closed
