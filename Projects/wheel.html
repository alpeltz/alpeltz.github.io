Wheel of Fortune

Practical Programming in Java
Created a wheel of fortune game simulation, which made use of Java’s GUI interface. This simulation allowed the user to spin the wheel and guess the letters to solve a puzzle, and allowed for multiple “players” in a game.
Main concepts implemented:
  • Use of Java’s GUI interface, along with using the layout options (Flow, box, grid) to recreate a UI interface that was specified in an image
  • User input through button clicks + event listeners
  • Updating UI components based on events in the code
  • Java Dialogs to present the user with information in a popup window
  • Implementations of custom Exceptions
