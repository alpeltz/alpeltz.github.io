Calculator

Introduction to Programming and Data Structures
Implemented a command-line stack based calculator, which uses postfix notation to parse and execute arithmetic commands. This is also referred to as "reverse Polish" notation. The arithmetic commands implemented included addition, subtraction, multiplication, and division.
Main concepts implemented:
  • Introduction to a stack data structure, with implementing our own push() and pop() functions
  • Input processing through command line
  • Creation of common data structures, such as List and Stack
  • Introduction to templating, I created a templatable Stack class so that it could be created with any other data type
  • Introduction to iterators, used for lists and other data structures
