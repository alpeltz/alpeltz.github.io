Error log manager

Data Structures and Algorithms
Created a command line error log manager. The log manager reads in a file of error logs and then allows the user to use a command line interface to filter by timestamp, keyword, id, category, and allows the user to append a new log entry.
Main concepts implemented:
  • Use map data implementations to store categories and keywords, where the key is the category/keyword and the value is a vector of log entry indices where the category or keyword occurs, allowing for an easy search for a specific value
  • Kept track of an excerpt list and search results in order to allow the user to append search results to the excerpt list for flexible searching
  • Usage of standard template library for some external functions
  • Implemented custom comparisons for ADTs and other classes, in order to apply sorting functions to them
