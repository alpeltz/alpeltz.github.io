Euchre

Introduction to Programming and Data Structures
Implemented a simulated Euchre card game through command line entry, with the implementation of human and simulated players. The game consisted of four players who each have a hand of cards, which they can then use to play Euchre by adding input to the command line. The game followed a subset of traditional Euchre rules, and the simulated players would have a specific playing algorithm.
Main concepts implemented:
  • Classes and class inheritance
  • Introduction to virtual/base class
  • Introduction to factory classes
  • Implemented custom sorting functions
  • Pointer new/delete allocation, dynamic objects
  • Implemented a complex set of rules for a card game from real life, by implementing rules and basic decision algorithms for simulated players
