Introduction to Machine Learning
Used feature matrices to train a classifier on a set of reviews. The reviews had a positive or negative label, and the classifier was trained to have parameters that would divide the positive and negative training data points as best as possible. Then it would be used to classify new, unlabeled reviews as either positive or negative.
Main concepts implemented:
  • Understanding of feature matrices and how to populate them
  • Used stratified k-fold cross validation so that a specific data set did not get overfit
  • Used grid search and random search iterations in order to iterate through different parameter values to find the most optimal ones for the base classifier
  • Used pre-existing SVC ‘fit’ and ‘predict’ functions that were already included in the sklearn library in order to fit models
  • Used different performance metrics and understood the differences of each one, including AUROC
