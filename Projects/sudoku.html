Sudoku Solver

Introduction to Artificial Intelligence
Used search algorithms in order to find the solution to a well known puzzle, Sudoku. The algorithm implemented backtracking with AC-3 (arc consistency) in order to prune "bad" solutions. For example, in Sudoku, you are able to analyze if choosing one value for a square will remove all possibilites for a different square, making the partial solution bad, and allowing you to prune it.
Main concepts implemented:
  • Search algorithms using backtracking and arc consistency in order to find a solution faster than brute force/just backtracking
  • Keep track of domains for certain variables to ensure a partial solution is viable
  • Understanding of DFS search algorithm in general, but implements pruning in order to make sure that ‘bad’ paths don't get continued
