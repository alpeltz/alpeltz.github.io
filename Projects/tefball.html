Ball game simulation

Practical Programming in Java
Created a simulated “tefball” game through a command line interface. The user can enter the game field through a text file, and the program will simulate the game until it is complete, updating the user with an updated version of the field. The field consisted of "." characters for empty spaces, and "R", "Q", "D" values for "Reciever", "Quarterback", "Defender" for the different types of players. Class inheritance was implemented to create different behavior for different players.
Main concepts implemented:
  • Object inheritance
  • Custom enums
  • Dynamic object creation with “new” and understanding of Java’s automatic garbage collection
  • Understanding of Get() and Set() methods in order to protect private fields of a class
