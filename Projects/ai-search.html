Search Algorithms

Introduction to Artificial Intelligence
I created search agents that were able to find the solution to a maze quickly, and with multiple different search algorithms. We were able to visualize the maze solving with a graphical 'pacman' interface that allowed the visualization of the algorithms to gain a deeper understanding. Basic breadth-first and depth-first search algorithms were explored at the start, and then I introduced iterative deepening, and an understanding of heuristics with A* search.
Main concepts implemented:
  • Basic BFS and DFS search implementations
  • Iterative deepening (DFS with an incrementally increasing depth limit)
  • Uniform cost search, using a priority queue to search nodes with lowest cost first
  • A* search, which then allowed an understanding of heuristics
